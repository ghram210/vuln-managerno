# كيفية تحميل قاعدة بيانات NVD كاملة على Kali Linux

لضمان ظهور كافة الثغرات، خاصة للبرمجيات القديمة (مثل Apache 2.4.7 الذي ظهر في فحوصاتك)، نوصي بتحميل أرشيف NVD كاملاً من عام **2002** وحتى اليوم.

## الخطوات:

1. افتح شاشة التيرمنال (Terminal) داخل مجلد المشروع.
2. قم بتشغيل الأمر التالي:

```bash
python3 scripts/download_nvd.py --start-year 2002
```

## لماذا عام 2002؟
* البرمجيات القديمة التي تستخدمها بعض المواقع (مثل scanme.nmap.org) تملك ثغرات تم اكتشافها في 2013 أو 2014.
* إذا حملت القاعدة من 2017 فقط، فلن يجد النظام هذه الثغرات القديمة وسيعتمد على "التصنيف الذكي" فقط.
* تحميل القاعدة من 2002 يضمن ظهور كلمة **OFFICIAL VULNERABILITY** لأي برنامج مهما كان قديماً.

## ملاحظات هامة:
* السكربت ذكي: لن يعيد تحميل السنوات الموجودة لديك (مثل آخر 3 سنوات)، بل سيكمل النقص فقط.
* المساحة المطلوبة: حوالي 5 جيجابايت.
* الوقت: حوالي 45 دقيقة للتحميل الكامل لأول مرة.

---
**النظام الآن مبرمج ليعطي الأولوية القصوى للقاعدة (NVD)؛ فإذا وجد تطابقاً، سيلغي أي تصنيف ذكي ويأخذ التصنيف الرسمي فوراً.**
