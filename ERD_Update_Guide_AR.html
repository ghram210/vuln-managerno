دليل تحديث مخطط الـ ERD في برنامج StarUML

1. إضافة حقل (Column/Attribute) جديد

  • اضغط بزر الفأرة الأيمن على الجدول، اختر Add ثم Column.
  • اكتب اسم الحقل ونوعه، مثل: raw_output: text.

2. محتويات جدول nvd_cves (المرجع العالمي)

  • id: uuid (PK) - المعرف الفريد للسجل.
  • cve_id: text (Unique) - الرقم التعريفي العالمي للثغرة.
  • description: text - الشرح التقني للثغرة.
  • cvss_score: real - درجة الخطورة (من 0 إلى 10).
  • severity: text - مستوى التصنيف (Critical, High, etc).

3. محتويات جدول exploits (أكواد الاستغلال)

  • id: uuid (PK) - المعرف الفريد.
  • exploit_id: text - المعرف في قاعدة Exploit-DB.
  • cve_id: text (FK) - الربط مع جدول الثغرات.
  • file_path: text - مسار ملف البرمجية الاستغلالية.

4. تحديث الجداول الحالية

  • في scan_results: أضف raw_output: text و options: text.
  • في vulnerabilities: أضف exprt_rating: text و vulnerability_count: integer.
